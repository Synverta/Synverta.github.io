如何修改这页内容

  1. 登录 GitHub → 打开仓库 Synverta/Synverta.github.io
  2. 点击页面上方的 index.html，然后点右上角的 Edit（铅笔图标）
  3. 修改文字后向下滚动，写一条 Commit 信息（比如 "update homepage"），然后点击 Commit changes
  4. 打开 https://synverta.github.io （大约等 1 分钟）即可看到更改

提示：如果页面没有出现，确认仓库名是 Synverta.github.io（注意大小写无关但拼写要对）。
